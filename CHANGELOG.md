# Changelog

All notable changes to Rowpack are documented here. The project follows [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2026-07-23

### Added

- CSV, TSV and JSON compilation into one self-contained HTML app.
- Editable grid, draggable Kanban and gallery views.
- Search, sorting, record creation, duplication and deletion.
- Undo and redo history with keyboard shortcuts.
- Direct HTML saving with a download fallback.
- Safe CSV export and automatic field-type inference.
- Responsive mobile layout and reduced-motion support.
- Strict Content Security Policy with zero runtime network requests.
- Unit, desktop and mobile end-to-end test suites.
